## What to build

Set up the foundation for Firebase-backed online mode. The app should be able to initialize the replacement Firebase project, sign in as an anonymous player, and run online-mode checks against a documented local or emulator setup without changing local 1 on 1 or AI mode behavior.

## Acceptance criteria

- [ ] The app can use replacement Firebase configuration for online mode.
- [ ] Anonymous Firebase Auth is enabled in the client flow.
- [ ] Online mode can identify the current anonymous player.
- [ ] Local 1 on 1 mode still starts and accepts moves.
- [ ] AI mode still starts and accepts moves.
- [ ] A documented verification path exists for the Firebase online-mode foundation.

- None — can start immediately.
